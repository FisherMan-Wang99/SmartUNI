## Introduction
This site is designed to help students discover their best-fit universities by filling a questionaire. 

### Data Source
The underlying data is scraped and restructured from several authoritative websites, including:
* **QS** World University Rankings
* **U.S. News** & World Report
* And other global education platforms.
